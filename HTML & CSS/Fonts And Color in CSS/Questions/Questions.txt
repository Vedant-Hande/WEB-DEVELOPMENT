Question 1: Customize Fonts and Colors for a Webpage
Task:

Create a simple webpage with an <h1>, <h2>, and two <p> elements.
Use a Google Font of your choice for the <h1> and a different font for the <p> elements.
Set the following styles:
h1 should have a font size of 3rem, be bold, and use a hex color.
h2 should be italicized and use an RGB color.
The <p> elements should have a line height of 1.6 and use a color with the rgba format.
Example Output: The headings should have distinct styles, and the paragraphs should be readable with good line spacing and color contrast.
